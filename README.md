#Dulce Saudades - Sitio Web

## 📌 Descripción

Este proyecto es una página web inspirada en la tradición brasileña del brigadeiro, diseñada para presentar una marca de repostería artesanal con una estética elegante y moderna.

La sección principal destaca la historia, calidad y autenticidad del producto, junto con una llamada a la acción para invitar a los usuarios a explorar los sabores disponibles.

---

## 🎯 Objetivo

Crear una experiencia visual atractiva que:

- Resalte la tradición brasileña
- Transmita calidad artesanal
- Invite al usuario a descubrir y comprar productos
